# Пример блога на node.js (Express) (Архитектура MVC)

Реализована панель администрирования, добавление/редактирование/просмотр статей.

Стек технологий:

* Nginx
* Node.js (Express)
* MongoDB
* Redis
* HTML5/CSS3 (Framework Bootstrap)
* Vanilla JavaScript
